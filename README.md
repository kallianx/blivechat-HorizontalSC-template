# Blivechat 横向SC HTML模板 - Horizontal SC HTML template

基于Blivechat默认模板修改的横向SC列表模板，支持点击变灰代表已读，适用于原版Blivechat。

**[Blivechat](https://github.com/xfgryujk/blivechat)**


## 安装
⚠注意⚠ 本项目基于BliveChat自定义HTML功能，请确保您将使用v1.10.1版本以上的BliveChat。

1. 找到 `BliveChat` 文件夹位置并打开目录 `data\custom_public\templates`
2. 下载并解压`HorizontalSC`至上述文件夹

解压完成后目录应该是这样的：

`data\custom_public\templates\HorizontalSC`

该目录下应当包含`index.html`文件

3. 启动`Blivechat`主程序，在`首页`的`自定义HTML模板`中可以找到`HorizontalSC`模板，选中后即可使用

Horizontal SC list template for original Blivechat, modified from the default template, featuring click-to-grey functionality.

## Installation
⚠️ Note ⚠️ This project is based on BliveChat's custom HTML functionality. Make sure you are using BliveChat version v1.10.1 or above.

1. Locate the `BliveChat` folder and navigate to the directory: `data\custom_public\templates`
2. Download and extract the `HorizontalSC` folder into the folder above

The folder structure should appear as follows:

`data\custom_public\templates\HorizontalSC`

This should contain an index.html file

3. Launch `BliveChat`, find the `HorizontalSC` template under the Custom HTML Templates section on the homepage. Select it to begin using the template.
